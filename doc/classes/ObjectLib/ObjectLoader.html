Class ObjectLib::ObjectLoader
In: lib/objectLoader.rb
Parent: Object

Takes a .obj file and converts it into C++ code suitable for embedding directly into an OpenGL C++ app. Capable of handling:

  • Vertex coordinates
  • Normals
  • Faces
  • Texture Coordinates
